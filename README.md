# Simple-LLM-Chatbot-Using-OpenAI-AP
This is a beginner-friendly chatbot built using the OpenAI Chat Completions API.  
It takes user input from the terminal and generates responses using the **GPT-4o-mini** model.

---

##  Features
- Uses OpenAI's latest chat-completions API  
- Clean and simple function (`ask_ai`) to call the model  
- Interactive chat loop in terminal  
- Demonstrates API usage, JSON parsing, and LLM integration
